  1. User startet Projekt und klickt "New Project"
  2. Runtime-Auswahl zeigt nur "JavaScript/Web (WebContainer)" - Pyodide entfernt
  3. Nach Projekt-Erstellung öffnet sich die IDE-Ansicht
  4. Chat-Interface ist verfügbar mit 3 Tabs:
    • 💬 KI Chat - Für Prompt-Eingabe
    • 🌐 WebContainer - Zeigt die laufende App
    • 📺 Console - Zeigt WebContainer-Output
  5. User kann einen Prompt eingeben (z.B. "Erstelle eine To-Do-App")
  6. Prompt wird an AJAX-Endpunkt gesendet und in DB gespeichert
  7. WebContainer startet automatisch und zeigt "Hello World" App
  8. App läuft in echtem Container mit HTML/CSS/JS
